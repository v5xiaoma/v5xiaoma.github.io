sql常用语句整理

发表于 2016-05-28   |   分类于 学习 ， 数据库 ， mysql   |     |   阅读次数

MySQL是一个关系型数据库管理系统，由瑞典MySQL AB 公司开发，目前属于 Oracle 旗下公司。MySQL 最流行的关系型数据库管理系统，在 WEB 应用方面MySQL是最好的 RDBMS (Relational Database Management System，关系数据库管理系统) 应用软件之一。

创建删除数据库和表

  • 创建数据库 create database 数据库名 charset=utf8;
  • use 数据库名; 进入数据库使用
  • 创建表 create table 表名(键名 类型,键名 类型);
  • 查看数据库/表 show databases/tables;
  • 查看表结构 desc 表名;
  • 表中插入数据 insert into 表名 set 键名='值',键名='值'...;
  • 多条数据插入 insert into 表名(键名,键名,...) values('值','值',...),(...);
  • 删除数据库或表 drop database 数据库名/表名 if exists;
  • 复制表结构到新表 create table 新数据库名 like 旧数据库名;
  • 复制表数据到新表 insert into 新数据库名 select * from 旧数据库名;
  • 赋值表结构和数据到新表 create table 新数据库名 select * from 旧数据库;

表键结构类型

  • 小数据类形/数据类形 tinyint/int
  • 字符串型/会随内容改变大小的字符串型 char/varchar
  • 特殊 unsigned(无符号数)/primary key(主键)/auto_increment(自增)/unique(唯一)/default(默认值)/zerofill(自动填充0)
  • 类似单选 enum('男','女');
  • 类似多选 set('1','2','3','4');

数据查询

  • 查询数据库表中内容 select * from 表名;
  • 查询数据库表中name条对应数据，并更名newname select name as newname from 表名;
  • 查询表中符合条件 和/或 条件的数据 select * from 表名 where 条件 and/or 条件;
  • 连接键名输出 select concat('连接内容',键名) from 表名;
  • 去掉重复的数据 select distinct 键名 from 表名;
  • 查询set类型字段 select * from 表名 where find_in_set('推荐' ,存入的名);
  • 查询set类型字段 select * from 表名 where flag like ‘%置顶%’
  • 查询set类型字段 select * from 表名 where flag='推荐,图文';
  • 查找 字段值是null 的 select * from 表名 where title is null;
  • 字段值为空的数据并返回空 select 键名,ifnull(age,"空") from 表名;
  • 排序 降序排列/升序排列（默认） select * from 表名 order by 排序依据的键名 desc/asc;
  • limit(m,n) 取出从m开始的n条数据， m以0开始记数 select * from 表名 order by 排序依据的键名 desc limit m,n;
  • between 值1 and 值2在值1与值2区间内检索内容 select * from 表名 where 判断的字段名 between m and n;
  • In(值1,值2….) 检测检索值是否在in表达式指定的值当中 select * from 表名 where 判断的字段名 in(12,20,22);
  • 查找名字当中 包含a的同学 select * from 表名 where sname like '%a%';
  • left从左侧取字符串截取学生名字的前n位 select left(查找字段,n) as 新名 from 表名;
  • mid 从中间取字符串,参数n为起始字符参数m为长度从中间截取 第一个参数 从几号开始（起始值1），第二个参数 截几位 select mid(查找字段,n,m) from 表名
  • rand()随机数//随机找n个同学 select * from 表名 order by rand() limit n;

多表查询

  • 关联 表时 使用别名 select * from city as c inner join zone as z on c.zid = z.zid;
  • 自连接 select s2.username,s1.username from student s1 join student s2 on s1.class=s2.class where s1.username ='李磊';

数据库修改操作

  • replace如果主键sid 不存在13，执行插入操作。主键存在sid=13 执行修改操作 replace into 表名(字段,字段,...) values(值1,'值2',...);
  • update 更新操作//注意：要加where条件，否则会更新表中全部数据 update 表名 set 字段='新值' where 条件(选择哪一条修改);
  • delete 删除操作//注意：要加where条件,否则会删除表中全部数据 delete from 表名 where 条件(选择哪一条删除);
  • alter table 表名 rename 新表名 alter table 表名 rename 新表名;
  • 修改字段类型 alter table 表名 modify 字段名 存储类型;
  • 修改字段名，并且更名，first字段放到第一，after 字段放到那个字段后面 alter table 表名 change 旧字段 新字段 存储类型 [ first | after 字段]
  • 追加字段 alter table 表名 add 字段 存储类型 [first | after 字段]
  • 删除字段 Alter table 表名 drop 字段名
  • 删除主键,需先去掉自增属性 alter table s modify 字段名 存储类型（不写auto_increment）;alter table s drop primary key;
  • 添加主键id alter table cb add primary key (id);

分组统计

  • count 统计数量 select count(要统计的字段) from 表名;
  • min () 取最小值 select min(要取最小值的字段) from 表名;
  • max() 取最大值 select max(要取最大值的字段) from 表名;
  • sum()计算和 select sum(要求和的字段) from 表名;
  • avg()求平均数 select avg(要求平均数的字段) from 表名;
  • 分组统计 select class,count(*) from student group by 用什么分组;
  • having在分组之后 select * from student where money>10000 group by class having count(*)>1;
  • 关键字顺序 1,where 2,group by 3,having 4,order by 5,limit

mysqli连接数据库

  • 查看php配置项信息 phpinfo();
  • 连接数据库，用$mysqli接收@new mysqli(‘主机名’,’用户名’,’密码’,’数据库’); $mysqli = @new mysqli('localhost','root','','ma');
  • connect_error数据库里面的函数用于记录错误信息 if($mysqli->connect_errno){echo $mysqli->connect_error;}
  • 设置字符集为utf8 query()执行里面的sql语句 $mysqli->query('SET NAMES UTF8');
  • 执行sql语句操作 query()执行里面的sql语句 $mysqli->query('SELECT * FROM stu');
  • affected_rows获取受影响条数 $mysqli->affected_rows();
  • insert_id获取新增的主键,修改删除不受影响,返回值为0; $mysqli->insert_id();
  • $result->fetch_assoc()取出数据库中的一条结果;$result是获取的执行结果query; $arr=array();while($a=$result->fetch_assoc()){$arr[]=$a;}

php操作mysql

  • mysql_connect()建立数据库连接 $conn = @mysql_connect("localhost", "username", "password") or die("不能连接到Mysql Server"); 说明：使用该连接必须显示的关闭连接
  • mysql_pconnect()建立数据库连接 $conn = @mysql_pconnect("localhost", "username", "password") or dir("不能连接到MysqlServer"); 说明：使用该连接函数不需要显示的关闭连接，它相当于使用了连接池

  • mysql_close()关闭数据库连接

  • mysql_select_db()选择数据库 @mysql_select_db("MyDatabase") or die("不能选择这个数据库，或数据库不存在");

  • mysql_query()查询MySQL $query = "select * from MyTable";$result = mysql_query($query); 说明：若SQL查询执行成功，则返回资源标识符，失败时返回FALSE。若执行更新成功，则返回TRUE，否则返回FALSE
  • mysql_db_query()查询MySQL $query = "select * from MyTable";$result = mysql_db_query("MyDatabase", $query); 说明：为了使代码清晰，不推荐使用这个函数调用

  • mysql_result()获取和显示数据

    说明：最简单、也是效率最低的数据获取函数

  • mysql_fetch_row()获取和显示数据

    说明：函数从result_set中获取整个数据行，将值放在一个索引数组中。通常会结使list()函数使用

  • mysql_fetch_array()获取和显示数据

    说明：result_type的值有：MYSQL_ASSOC: 字段名表示键，字段内容为值MYSQL_NUM: 数值索引数组，操作与mysql_fetch_ros()函数一样MYSQL_BOTH: 即作为关联数组又作为数值索引数组返回。result_type的默认值。

  • mysql_fetch_assoc()获取和显示数据 array mysql_fetch_assoc (resource result_set)相当于调用 mysql_fetch_array(resource, MYSQL_ASSOC);

  • mysql_fetch_object()获取和显示数据

    说明：返回一个对象，在操作上与mysql_fetch_array()相同

  • mysql_num_rows()所选择的记录的个数

    说明：只在确定select查询所获取的记录数时才有用。

  • mysql_affected_rows()
